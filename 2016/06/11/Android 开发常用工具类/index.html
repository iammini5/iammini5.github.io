Android 开发常用工具类

发表于 2016-06-11   |   分类于 Android   |     |   本文总阅读量 次

后续将封装更多的工具类，将尝试封装一些跟View相关的工具类，像Adapter….,功能将不断完善，同时感谢每一位开源者的贡献，本工具类参考了很多网上工具类。
封装一些常用的工具类，目的是让程序猿有更多自由时间，可以去看电影，撩妹纸。

1.由于水平有限,难免有Bug,如果有任何问题,请反馈 xyb520zly@gmail.com
2.如果你有更好的代码,请提交Pull request.
3.感谢各位的star,你们的支持是我继续前进的动力.
4.项目地址：https://github.com/xybCoder/AndroidCommon

SPUtil SharedPreferences工具

  1. getSpName(Context context) 获取SharedPreferences存储在sd卡中的文件名字
  2. setSP(Context context, String key, Object object) 根据传入不同参数类型调用不同的保存方法
  3. getSP(Context context, String key, Object defaultObject) 根据默认值得到保存的数据的具体类型，然后调用对应方法获取值
  4. removeSP(Context context, String key) 移除某个key值已经对应的值
  5. clearAllSP(Context context) 清除所有SharedPreferences数据
  6. contains(Context context, String key) 查询某个key是否存在
  7. getAllKeyValue(Context context) 返回所有的键值对

DateUtil 时间工具

DateUtils类里面重载方法比较多

  1. str2Date(…) 将时间字符串转换成Date
  2. date2Str(…) 将Date转换成时间字符串
  3. str2Calendar(…) 将时间字符串转换成Calendar
  4. getCurDateStr(…) 获得当前日期的字符串
  5. getMillon(…) 获得当前日期的字符串,精确到秒
  6. strAddOneDay(…) 实现加一天的功能，返回的格式为String
  7. strDecreaseOneDay(…) 实现减一天的功能，返回的格式为String
  8. getLastdayDate(…) 获取昨天 Data
  9. getNextdayDate(…) 获取明天Date
  10. isTheSameDay(…) 判断是否是同一天

LogUtil 日志工具

  1. init 初始化日志开关和TAG(默认日志为开,TAG为”ghost”)
  2. v VERBOSE
  3. d DEBUG
  4. i INFO
  5. w WARN
  6. e ERROR
  7. a ASSERT
  8. json 输出json
  9. xml 输出xml

    StringUtil 字符串工具

  10. isEmpty(CharSequence str) 判断是否是空字符串、null、””
  11. subStringOmit(String subject, int size) 字符串截取到指定长度size+…的形式
  12. getLimitLengthStr(String str, int len, String symbol)截取字符串　超出的字符用symbol代替
  13. getStrLength(String str)获取字符串的实际长度（考虑了汉字的情况一个汉字按照俩个字符算）
  14. getHideEmailPrefix(String email)隐藏邮件地址前缀
  15. Str2List(String str,String symbol)字符串分割成一个数组
  16. List2Str(List list, String symbol)将list 用传入的分隔符组装为String
  17. replaceBracketStr(String str) 全角括号转为半角
  18. full2Half(String str)全角字符变半角字符
  19. replaceBlank(String str)去除字符串中的空格、回车、换行符、制表符
  20. str2Unicode(String string)字符串转换unicode
  21. unicode2Str(String unicode)unicode 转字符串
  22. trimPunct(String str)删除所有的标点符号
  23. SimilarDegree(String str1, String str2)字符串相似度比较
  24. getStrEncode(String str)获取字符串的编码
  25. countSubStr(String string, String str)获取字符串str在String中出现的次数
  26. nullToEmptyStr(Object str)空对象转化为空串
  27. capitalizeFirstLetter(String str)首字母大写

  28. utf8Encode(String str)str使用UTF-8编码

    GsonUtil Gson工具

  29. object2Json(…)object转json
  30. json2List(String jsonStr) jsonStr转List
  31. json2Map(String jsonStr)jsonStr转Map
  32. json2Bean(String jsonStr, Class<?> cl) jsonStr转Bean
  33. getJsonValue(String jsonStr, String key) 获取key值

VerificationUtil 验证工具类

  1. matcherRealName(String value) 判断姓名格式

    1.真实姓名可以是汉字,也可以是字母,但是不能两者都有,也不能包含任何符号和数字
    2.如果是英文名,可以允许英文名字中出现空格
    3.英文名的空格可以是多个,但是不能连续出现多个
    4.汉字不能出现空格

  2. matcherPhoneNum(String value) 判断手机号格式 (匹配11数字,并且13-19开头)

  3. matcherAccount(String value) 判断账号格式 (4-20位字符)
  4. matcherPassword(String value) 判断密码格式 (6-12位字母或数字)
  5. matcherPassword2(String value) 判断密码格式(6-12位字母或数字,必须同时包含字母和数字)
  6. matcherEmail(String value) 判断邮箱格式
  7. matcherIP(String value) 判断IP地址
  8. matcherUrl(String value) 判断URL (http,https,ftp)
  9. matcherVehicleNumber(String value) 判断中国民用车辆号牌
  10. matcherIdentityCard(String value) 判断身份证号码格式
  11. isNumeric(String value) 是否数值型
  12. testRegex(String regex, String inputValue) 是否匹配正则
  13. checkPostcode(String postcode) 匹配中国邮政编码
